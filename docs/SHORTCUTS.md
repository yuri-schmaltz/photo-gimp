# PhotoGIMP Shortcuts (Photoshop-like)

This file lists key shortcuts aligned with common Photoshop habits. You can customize more inside GIMP (`Edit > Keyboard Shortcuts`).

| Action | PhotoGIMP Shortcut | Photoshop Equivalent | Notes |
| --- | --- | --- | --- |
| Save | Ctrl/Cmd + S | Ctrl/Cmd + S | Quick save
| Save As | Ctrl/Cmd + Shift + S | Ctrl/Cmd + Shift + S | Overwrite prompt
| Undo | Ctrl/Cmd + Z | Ctrl/Cmd + Z | Step back
| Redo | Ctrl/Cmd + Shift + Z | Ctrl/Cmd + Shift + Z | Step forward
| Duplicate layer | Ctrl/Cmd + J | Ctrl/Cmd + J | Creates a copy of the active layer
| New layer | Ctrl/Cmd + Shift + N | Ctrl/Cmd + Shift + N | Opens new layer dialog
| Fit on screen | Ctrl/Cmd + 0 | Ctrl/Cmd + 0 | Zoom to fit
| Close image | Ctrl/Cmd + W | Ctrl/Cmd + W | Closes active document
| Scale/Transform | Ctrl/Cmd + T | Ctrl/Cmd + T | Mapped to GIMP Scale tool

## How to extend

- Open GIMP > Edit > Keyboard Shortcuts and add/change bindings.
- The raw configuration is stored in `.config/GIMP/3.0/shortcutsrc`.
- After editing in GIMP, copy your updated `shortcutsrc` back to this repository to share with others.
